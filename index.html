ARTIC API Project

About

This is a coding project that pulls datapoints from the Art Institute of Chicago's open API about cats and dogs in art and returns the title and image of the first ten entries on random pages upon clicking the navigation links.

The sampling has been limited to the first five pages. Click on the navigation links at the top to "get" the API artwork data!
